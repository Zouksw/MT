# 数据源全链路审计报告

**审计日期**: 2026-05-14（修订版）
**服务器位置**: 中国大陆
**核心发现**: 之前的审计错误地将大量中国源标记为"地理封锁"——实际原因是中国政府网站改版/API路径变更、代码bug、和缺失API Key

## 根因分析：为什么数据源没有打通

### 第一类：代码 Bug（最冤——端点有数据但代码吃掉了）

| 数据源 | 端点状态 | 代码Bug | 影响 |
|--------|---------|---------|------|
| **cme_futures** (Stooq) | **200，14个symbol全部有数据** | `if (lines.length < 2) return null;` — Stooq只返回1行CSV（无header行），永远 `return null` | **14个期货品种**全部无数据 |

**详细说明**: Stooq API 返回格式为 `LE.F,20260512,230000,245.75,247.15,240.75,241.075,,\r\n`，仅1行。代码第139行判断 `lines.length < 2` 导致永远跳过。这是**最大的单一bug**，修复后立即获得14个期货品种数据。

### 第二类：API Key 缺失（端点可达但认证失败）

| 数据源 | Key环境变量 | 端点状态（无Key） | 获取难度 | 影响 |
|--------|------------|------------------|---------|------|
| **fred** | FRED_API_KEY (**空**) | 400 (Bad Request) | 免费，即时获取 fred.stlouisfed.org | 经济指标 + baltic_dry 备源 |
| **usda_ams** | USDA_MARS_API_KEY (**空**) | 403 (Forbidden) | 免费，需注册 marsapi.ams.usda.gov | 美国牛肉分级数据 |
| **weather** | OPENWEATHER_API_KEY (**空**) | 401 (Unauthorized) | 免费，即时获取 openweathermap.org | 8个气象站数据 |
| **mla_nlrs** | MLA_API_KEY (**空**) | 需认证 | 需MLA账户 | 澳洲牛肉出口数据 |
| **baltic_dry** | 依赖FRED_API_KEY | 主源404，备源无Key | 同FRED | BDI航运指数 |

### 第三类：中国网站改版/API路径失效（非地理封锁）

| 数据源 | 端点状态 | 真实原因 | 影响 |
|--------|---------|---------|------|
| **china_wholesale** (moa) | 主源404，备源502 | 农业部网站改版为Vue SPA，旧API路径 `/product/price-info/getPriceInfoList` 已不存在。根域名可达(200)，新前端为 `ncpscxx.moa.gov.cn` | **11个中国农产品品种** |
| **dce_futures** | 412 (反爬) | 大商所部署了瑞数信息(Ruishu)反爬系统，HTTP请求返回JS挑战页面。**不是地理封锁，是反爬** | 6个大商所期货 |
| **shipping_index** (SSE) | 404 (Page Not Found) | 上海航运交易所网站改版，`/index/scfi` 路径已不存在，返回 "Page Not Found" | SCFI/CCFI航运指数 |
| **china_customs_stats** | 连接超时(000) | CDN解析问题（CNAME指向 365cyd.cn），可能需要特定DNS或内网环境 | 中国牛肉进口数据 |

### 第四类：海外站点从中国访问受限（真正的地理/反爬问题）

| 数据源 | 端点状态 | 真实原因 | 影响 |
|--------|---------|---------|------|
| **cepea** (巴西) | 403 | 巴西学术站点拒绝非巴西IP（CloudFront 403） | Boi Gordo 牛肉价格 |
| **inac** (乌拉圭) | 超时(000) | 乌拉圭政府站点对中国IP响应不稳定 | 乌拉圭牛肉出口价格 |
| **abares** (澳洲) | 超时(000) | 澳洲政府站点对中国IP响应不稳定 | 澳洲牛肉生产/出口统计 |
| **secex** (巴西) | 403 | ComexStat API 端点可达但拒绝请求（可能需要特定的headers/origin） | 巴西牛肉出口数据 |

### 第五类：API端点已下线

| 数据源 | 端点 | 状态 | 说明 |
|--------|------|------|------|
| **world_bank** | `api.worldbank.org/v2/commodity` | **404** (XML错误页) | 世界银行已下线 Commodity Price API（Pink Sheet API），但 Pink Sheet 数据页面仍存在 (`thedocs.worldbank.org`)。v2/country 等其他API正常。 |
| **usda_psd** | `apps.fas.usda.gov/psdonline/api/data/commodity` | **404** | API路径已变更，但网站首页可达 (`apps.fas.usda.gov/psdonline/app/index.html` → 200) |
| **fao_prices** | `fenixservices.fao.org/faostat/api/v1/en/data/CP` | **521** (超时) | FAO服务降级，可能是临时问题 |

### 第六类：Stub / 未实现

## 真实问题优先级（按投入产出比排序）

### 立即修复（1小时内，影响最大）

1. **修复 CME/Stooq 代码Bug** — 改 `lines.length < 2` 为 `lines.length < 1`
   - 立即解锁 **14个期货品种** 的每日数据
   - 这是当前ROI最高的修复

2. **申请 FRED API Key** — 免费，即时获取
   - 解锁 FRED 经济指标 + Baltic Dry Index 备源

3. **申请 OpenWeather API Key** — 免费，即时获取
   - 解锁 8个气象站数据

### 中等投入（2-4小时）

4. **修复 World Bank 数据获取** — Pink Sheet页面可达(200)，改用HTML抓取或查找新API
   - 恢复覆盖 **37个品种** 的最大单一数据源

5. **修复 USDA PSD API 路径** — 网站首页可达，需找到新API路径
   - 解锁全球供需基本面数据

6. **申请 USDA MARS API Key** — 免费，需注册
   - 解锁美国牛肉分级/活牛价格数据

### 长期方案

7. **中国源改版适配** — 需要逆向工程新SPA接口
   - china_wholesale: 找Vue SPA的新API端点
   - shipping_index: 找SSE新版数据接口
   - china_customs: 解决DNS CDN问题

8. **DCE/CZCE反爬绕过** — 需要Puppeteer/Playwright或找到官方数据接口

9. **巴西/乌拉圭/澳洲源** — 可能需要代理IP或切换数据提供商
